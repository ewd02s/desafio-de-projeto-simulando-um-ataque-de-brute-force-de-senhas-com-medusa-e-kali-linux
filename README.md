# Desafio de projeto: Simulando um Ataque de Brute Force de Senhas com Medusa e Kali Linux

Implementei um projeto prático utilizando Kali Linux e a ferramenta Medusa, em conjunto com ambientes vulneráveis Metasploitable 2 e o DVWA, para simular cenários de ataque de força bruta. o que aprendi, com o estudo, reflexões e exemplos de código.

- Configuração do ambiente: duas VMs (Kali Linux e Metasploitable 2) no VirtualBox, com rede interna (host-only).
- Executei ataques simulados de força bruta em FTP, automação de tentativas em formulário web (DVWA) e password spraying em SMB com enumeração de usuários.
- Documentei testes: comandos utilizados, wordlists simples, validação de acessos e recomendações de mitigação.
